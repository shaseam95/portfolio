BMW TABLET WEBSITE

For my User Experience module at University, I designed a prototype tablet optimised version of the BMW website. I performed a usability test on the existing website following the CIF to discover issues to rectify in the redesign. One of the main issues found was that the website did not contain any product comparison tool, which made it difficult for users to compare the different car models. This was a new feature I implemented.

The website was made using HTML, CSS, JavaScript and jQuery. Some images where edited in Photoshop to remove their backgrounds.
Grade awarded: 74%.

VIEW RESTAURANT ORDERING SYSTEM

For this project I created a restaurant ordering system that digitalise the operations of processing orders in a restaurant in the form of a web application. The previous method of processing an order was paper based which had issues such as high long-term costs. The application is used to create, read, update and delete restaurant orders.

The technologies used to create this were HTML, CSS, PHP, MySQL, PhpMyAdmin and jQuery.
Grade awarded: 60%.
